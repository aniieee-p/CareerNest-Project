# CareerNest
🚀 CareerNest is a full-stack MERN job portal that connects job seekers and recruiters in one platform. Users can explore opportunities, apply for jobs, and track applications while recruiters can post jobs and manage candidates through an intuitive dashboard.

# 🌟 CareerNest – Smart Job Portal

🚀 **CareerNest** is a modern **MERN stack job portal** designed to connect **talented job seekers with recruiters** in a seamless and efficient way.  
It provides a clean interface where candidates can discover opportunities and companies can find the perfect talent.

This platform simplifies the hiring process by offering **job posting, application tracking, dashboards, and powerful search features**.

# ⚡ Quick Start

### 1️⃣ Clone the repository

```bash
git clone https://github.com/aniieee-p/CareerNest.git
cd CareerNest
```

### 2️⃣ Setup Backend

```bash
cd backend
npm install
cp .env.example .env
# Configure your .env file with required credentials
npm run dev
```

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
cp .env.example .env
# Configure your .env file
npm run dev
```

### 4️⃣ Access the Application

- **Frontend**: http://localhost:5173
- **Backend API**: http://localhost:3000

📖 **For detailed deployment instructions, see [DEPLOYMENT.md](./DEPLOYMENT.md)**
